# Java Card Collection Program

This project is a simple Java program that uses **Collection Interfaces** (Map, List, HashMap, ArrayList) to store all playing cards and allow the user to **search for all cards of a given symbol** (Spade, Heart, Diamond, Club).

It is designed for **Java external examination / academic assignment** and demonstrates:
- Use of Collections Framework  
- Storing and organizing data  
- Searching inside a Map  
- Taking user input  
- Basic Java console application
